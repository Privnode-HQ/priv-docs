## 使用方法

<Steps>
    <Step>
        ### 确保 Node.js 与 npm 已安装

        运行
        ```
        npm -v
        ```
        如果有输出，则可以下一步。
        如果报错未找到命令，请参考[ Node.js 安装指南](https://www.runoob.com/nodejs/nodejs-install-setup.html)。
    </Step>
    <Step>
        ### 一键配置

        运行
        ```
        npx @privnode/cli@latest auto
        ```
        随后遵循终端中的指南即可自动配置。
    </Step>
</Steps>

## 命令参考

### `install`

安装指定的 CLI 工具。

### `install-all`

安装所有可用的 CLI 工具。

### `upgrade`

升级指定的 CLI 工具。

### `upgrade-all`

升级所有可用的 CLI 工具。

### `uninstall`

解除安装指定的 CLI 工具。

### `uninstall`

解除安装所有可用的 CLI 工具。

### `auto`

特定 CLI 工具的交互式设置向导。

### `auto-all`

所有可用 CLI 工具的交互式设置向导。

### `doctor`

生成 CLI 运行环境诊断报告以作故障排除用途。
